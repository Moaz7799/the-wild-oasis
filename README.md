# 🌴 The Wild Oasis

A modern hotel booking dashboard built with React, styled-components, and Vite.

This app provides a clean and responsive interface to manage guests, cabins, bookings, and more — designed as an internal admin panel for managing a fictional hotel business called **The Wild Oasis**.

---

## 🚀 Features

- ✅ View and manage bookings, cabins, and guests
- 🌗 Dark mode support with context API
- 📆 Filter and sort bookings by date or status
- 🔍 Search and pagination
- 💅 Fully responsive design
- ⚛️ Built using React + styled-components
- ⚡ Fast bundling and development with Vite

## 🛠️ Tech Stack

- **React** — UI and components
- **React Router** — routing and navigation
- **Styled-components** — styling
- **Vite** — fast dev server and build tool
- **React Query / Context API** — (optional) for data/state management
- **Supabase / JSON server** — (optional) for backend API
